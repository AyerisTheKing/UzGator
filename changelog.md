Версия 5.1
- Добавлены новые триггеры логирования во всей системе, в том числе и переключение по вкладкам (switchTab). Теперь записываются: загрузка фото в галерею, скачивание карточки летописи, старт квиза (даже неоконченного), удаление лайка, модерирование фото админом, удаление записей администратором, а также все перемещения между вкладками и под-разделами аналитики.

Версия 5.0
- Добавлено окно профиля пользователя в Консоли (Логи): теперь можно просматривать подробную информацию по каждому участнику (ID, Класс, Статус, Был ли заблокирован) и полный список его последних действий.
- Исправлен баг в галерее: кнопки "Лайк" и "В закладки" больше не "улетают" вверх экрана при прокрутке.
- Исправлено отображение изображений в галерее: теперь горизонтальные картинки (формата 16:9) плавно вписываются в область 9:16 с сохранением пропорций, без грубого обрезания центра кадра.

Версия 4.5
- Добавлен раздел Консоль (Логи) в Админ-панель для отслеживания активности, статуса онлайн и текущей вкладки пользователей.
- Внедрена система логирования действий пользователя (`logAction`) и трекинг присутствия (`updatePresence`).
- Исправлены уязвимости UI-элементов во множестве форм (`handleOnboarding`, `adminAddQuiz`, `adminAddChronicle`): теперь кнопки защищены от двойных кликов с блокировкой (disabled), а все асинхронные блоки кода обернуты в try/catch для предотвращения падения приложения (ошибка кетчится в консоль, UI плавно восстанавливается).
- Добавлена задержка (debounce / блокировка dataset) на лайки.
